# 冒险插画响应式优化报告

## 选择结论

- 手机：720px 宽，WebP Q86，`sharp_yuv`，编码 method 6。
- iPad：1400px 宽，WebP Q88，`sharp_yuv`，编码 method 6。
- 桌面：保留原始宽度（1828-1983px），WebP Q90，`sharp_yuv`，编码 method 6。
- 纵向冒险地图只有 900 × 1500 的 WebP 源图。为避免二次有损和无意义放大，三端使用各自目录中的原图直拷版本。
- 全部原始 PNG 和原始地图均保留，未覆盖、未删除。
